# Royaume des Mygales — Package prêt à héberger

## Contenu
Fichiers principaux:
- index.html, elevage.html, gallery.html, especes.html
- styles.css, script.js, logo.svg
- assets/gallery/*.svg (images placeholders)

## Hébergement rapide (GitHub Pages)
1. Crée un dépôt GitHub public (ex: `royaume-mygales`).
2. Glisse tous les fichiers dans le dépôt (ou `git add` + commit + push).
3. Dans *Settings* → *Pages* → sélectionne la branche `main` et le dossier `/` (root).
4. Attends quelques minutes — ton site sera dispo sur `https://<username>.github.io/royaume-mygales/`.

## Hébergement facile (Netlify)
1. Crée un compte Netlify (gratuit).
2. Dépose le dossier `royaume_mygales_full` comme *drag & drop* sur Netlify Sites (Deploy site).
3. Netlify crée une URL publique instantanée.

## Personnalisation
- Remplace les images `assets/gallery/*.svg` par tes photos `photo1.jpg` etc.
- Ajoute un fichier audio dans `assets/audio/` nommé `forest.mp3` puis adapte le HTML si tu veux un lecteur.
- Le bouton "Télécharger en PDF" utilise `window.print()` — ouvre l'aperçu d'impression et enregistre en PDF.
